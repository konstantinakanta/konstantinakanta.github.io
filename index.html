06 Projects
Research

Pneumonia Detection from Chest X-rays

Compared four deep-learning architectures (logistic regression baseline, LeNet-5, custom CNN with data augmentation, VGG16) for binary pneumonia classification. Transfer learning from ImageNet with VGG16 performed best. Built in Python with scikit-learn, OpenCV, Keras, and PyTorch.

Research

Fusing Rankings in Information Retrieval with Voting Systems

Diploma thesis at DUTh. Evaluated rank fusion approaches (Borda, RRF, Condorcet, CombSUM/MNZ) on IR test collections using TREC Eval. RRF proved consistently strongest — the same method later drove a 10% accuracy gain at Shutterstock.

Code

Multi-label Classifier from Scratch

From-scratch implementation of Binary Relevance (with optional under-sampling) and Classifier Chains for multi-label classification. Built to internalise every design decision — thresholding, imbalance handling, label-correlation effects.

Code · Coming soon

rrf — Reciprocal Rank Fusion library

A small open-source Python package implementing RRF and related fusion methods (Borda, CombSUM, CombMNZ), plus a reproducible benchmark notebook. Motivated by RRF's recurring role — from my thesis through Shutterstock to modern RAG pipelines.
